MCP Tools

trustscore_check

Check a provider before you use it. Get trust score, reliability metrics, red flags, and performance history‚Äîall in one call.

// MCP tool call await mcp.call_tool("trustscore_check", { "provider_id": "github-integration-009" }); // Returns: {trust_score, reliability, // sample_size, confidence, flags}

trustscore_report

Report your results after each interaction. Your reports make scores more accurate‚Äîfor you and everyone else.

// Report outcome await mcp.call_tool("trustscore_report", { "provider_id": "github-integration-009", "outcome": "success", "latency_ms": 120 });

trustscore_rank

Compare multiple providers at once. Filter by task, set minimum thresholds, get ranked results instantly.

// Rank multiple providers await mcp.call_tool("trustscore_rank", { "providers": ["provider-a", "provider-b"], "min_score": 0.7 });

Frequently Asked Questions

Is Vetted free to use?
Yes, Vetted is 100% free and open source under the MIT License. You can install, use, and modify it without any cost or restrictions.
How accurate are the trust scores?
Scores are based on real agent reports from 9,300+ interactions. Each score includes a confidence level based on sample size. Scores with high confidence (100+ reports) are highly reliable.
How do I install Vetted?
Install via npm: npm install -g trustscore-mcp. Then add it to your MCP client config (Claude Desktop, Cursor, etc.). Full instructions in our GitHub README.
What data does Vetted collect?
We only collect anonymized reliability metrics (uptime, latency, error rates). No personal data, API keys, or request content is ever stored or transmitted.
Can I trust scores for new providers?
New providers start with a synthetic baseline score and show "Low Confidence" until they accumulate real reports. Always check the confidence level before making critical decisions.
How often are scores updated?
Scores update in real-time as new reports arrive. Trust decay algorithms automatically reduce scores for providers with aging data, ensuring freshness.
