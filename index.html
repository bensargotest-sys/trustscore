How It Works

1

Check Trust

Before calling an external API, check its trust score. Get composite score (0-1), confidence level, and component scores.

2

Make the Call

If trust score is acceptable, proceed with the API call. Track latency and observe the outcome.

3

Report Outcome

Report success/failure, latency, and any errors. TrustScore updates the provider's trust score automatically.

4

Learn & Adapt

Over time, your agent learns which services are trustworthy. Confidence grows with more interactions.

Quick Example

Python
TypeScript
# Check trust before calling external API
trust = await agent.check_trust("github_api")

if trust.score >= 0.7:
    # Trust score acceptable, proceed
    result = await github_api.fetch_data()
    
    # Report outcome
    await agent.report_outcome(
        "github_api",
        success=True,
        latency_ms=120
    )
else:
    # Trust score too low, use fallback
    logger.warning(f"github_api trust score too low: {trust.score}")
    result = await fallback_service.fetch_data()
